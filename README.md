# AVER Benchmark Leaderboard

Live results for the AVER (Agent Verification & Error Recovery) benchmark.

## About AVER

AVER measures AI agents' ability to:
- **Detect** errors (40% of score)
- **Diagnose** why errors occurred (20% of score)
- **Recover** and fix the problem (40% of score)

## Leaderboard

See live results on AgentBeats: [https://agentbeats.dev/benchmarks/aver](https://agentbeats.dev/benchmarks/aver)

## Categories

- **Hallucination**: Detected invented APIs/libraries
- **Validation**: Incorrect calculations or formats
- **Tool Misuse**: Wrong tool selection
- **Context Loss**: Forgetting constraints
- **Adversarial**: Ambiguous/conflicting requirements
